# A unique string
Given a string s, determine whether it has all unique characters.

### Example 1
Input: s = "abcde"

Output: true

Explanation: All characters only occur once

### Example 2
Input: s = "aab"

Output: false

Explanation: There's two as

### Example 3
Input: s = ""

Output: true

Explanation: All characters occur once (of which there are none)
